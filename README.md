# 🧠 Reto 30 Días: LLMs, Prompt Engineering & Agentes de IA aplicados a Inteligencia Financiera

Repositorio de evidencia de aprendizaje — Programa A ("Sprint por Capas")

## 🎯 Objetivo
Adquirir dos habilidades especializadas en 30 días:
1. **Fundamentos de LLMs + Prompt Engineering avanzado**
2. **Construcción de agentes y sistemas multi-agente aplicados a inteligencia financiera**

## 🗺️ Roadmap

| Semana | Enfoque | Entregable |
|---|---|---|
| 1 | Fundamentos de LLMs | Explicación documentada de context window, tokens, temperature |
| 2 | Prompt Engineering profesional | 10 prompts reescritos + script de evaluación |
| 3 | Agentes aplicados a inteligencia financiera | Agente funcional con casos de prueba |
| 4 | Multi-agente e integración final | Sistema multi-agente + demo + reporte |
