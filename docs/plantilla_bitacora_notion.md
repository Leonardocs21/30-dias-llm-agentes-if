# 📓 Plantilla de Bitácora — Reto 30 Días

Copia esta estructura en Notion. Sugerencia: crea una base de datos (database) con estas propiedades, y una página por día.

## Propiedades de la base de datos (columnas)

| Propiedad | Tipo | Opciones |
|---|---|---|
| Día | Número | 1-30 |
| Semana | Select | Semana 1 / 2 / 3 / 4 |
| Tema | Texto | — |
| Estado | Select | 🔴 Pendiente / 🟡 En progreso / 🟢 Completado |
| Horas invertidas | Número | — |
| Link commit GitHub | URL | — |
| Autoevaluación (1-5) | Número | — |

## Vista recomendada
Crea una vista "Board" agrupada por Estado, y otra "Timeline" o "Calendar" para visualizar tu constancia diaria — esto es muy útil como evidencia visual si algún día lo muestras en una entrevista.

## Página resumen (crear una sola, fija)
Una página aparte llamada "Resumen Ejecutivo del Reto" con:
- Las 2 habilidades que dominarás
- Link al repositorio de GitHub
- Un párrafo corto tipo "elevator pitch" que puedas reutilizar en LinkedIn o entrevistas describiendo qué hiciste en estos 30 días
